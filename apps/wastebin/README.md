## About Wastebin

**Wastebin** is a minimal self-hosted pastebin with a clean, modern interface inspired by bin. Built with Rust for excellent performance and low resource usage, it stores all entries in a SQLite database. Perfect for sharing code snippets, logs, and configuration files on your own infrastructure — without any third-party dependency.
