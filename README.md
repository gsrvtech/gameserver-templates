# gameserver-templates
Repo for Pelican & Jexactyl Eggs

## ⚠️ Pterodactyl Support Discontinued

Pterodactyl support for all **free eggs** has been officially discontinued.

Free eggs are now published as **Pelican YAML files only**. If you still need Pterodactyl versions, you have two options:

- **Download Pterodactyl versions** via our Discord bot using `/egg download` in our [Discord server](http://discord.gg/raurR4vshX)

> **Note:** Paid eggs are not affected — all paid releases continue to include both Pelican and Pterodactyl versions.

**Why?** Recent changes in Pelican Wings make full compatibility with Pterodactyl no longer reliable. Since everything is offered for free, maintaining dual support is no longer feasible.
