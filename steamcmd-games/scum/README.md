## About SCUM

**SCUM** is an open-world survival game set on a remote island that serves as the stage for a brutal reality show. You play as a prisoner fighting to survive — and boost your fame rating — against other players, mechanised puppets, and the environment. SCUM features an extraordinarily detailed character metabolism system, skill progression, crafting, base building, and large-scale PvP. Stay alive long enough and you might just make it out.
