## About Schedule I

**Schedule I** is an open-world drug dealer simulation game by TVGS. Start small, build your operation, manage supply chains, handle shady contacts, and expand your criminal empire — all while keeping a low profile. The game features a rich open world with detailed NPC routines and a progression system that rewards strategic thinking.

- **Steam**: [store.steampowered.com/app/3164500](https://store.steampowered.com/app/3164500/Schedule_I/)
- **SteamDB**: [steamdb.info/app/3164500](https://steamdb.info/app/3164500/)

## About this Egg

This repository contains **two variants** of the Schedule I dedicated server egg:

| Variant | File | Runtime |
|---------|------|---------|
| **Proton** | `egg-schedule-i-proton.yaml` | Proton (recommended) |
| **Wine**   | `egg-schedule-i-wine.yaml`   | Wine (fallback if Proton fails) |

The server is powered by the **S1DedicatedServerMod** (MelonLoader). You must use Proton or Wine because only the Windows binary is available.

> ⚠️ **Important:** A Steam account that **owns Schedule I** (App ID: 3164500) is required to download the game files.
