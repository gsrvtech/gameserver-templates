## About MotorTown: Behind The Wheel

**Motor Town: Behind The Wheel** is an open-world driving sandbox set in a vast, living city. Deliver goods, run a taxi service, race opponents, or just cruise the streets — whatever you want, whenever you want. The game features a huge map with diverse environments, a realistic physics model, and seamless online multiplayer.

- **Steam**: [store.steampowered.com/app/1369670](https://store.steampowered.com/app/1369670/Motor_Town_Behind_The_Wheel/)

## Download this Egg

This repository contains a **Proton** and a **Wine** egg.

> **You should use the Proton egg.** Some systems cannot run Proton — use the Wine egg as a fallback in that case.

These are **Pelican** YAML eggs.

> **Pterodactyl users:** You can get a Pterodactyl-compatible version in two ways:
> - 🔀 Convert it yourself with **[Scramble Egg Converter](https://redthirten.github.io/scramble-egg-converter/)**
> - 💬 Download a pre-built version via `/egg download` in our **[Discord server](http://discord.gg/raurR4vshX)**

---

## Important: SteamGuard

> ⚠️ You **MUST** disable SteamGuard completely (including email auth) before installing. SteamCMD cannot push authentication requests to a mobile device and you cannot enter email codes in the Pelican/Pterodactyl install console.
