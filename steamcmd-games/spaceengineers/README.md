## About Space Engineers

**Space Engineers** is a sandbox game set in space and on planets. It is an engineering, construction, exploration, and survival game where players build space ships, space stations, planetary outposts, and more. Every object in the game is fully destructible and can be used as material. Space Engineers focuses on creativity and engineering to survive in a hostile universe.

- **Steam**: [store.steampowered.com/app/244850](https://store.steampowered.com/app/244850/Space_Engineers/)
- **SteamDB**: [steamdb.info/app/244850](https://steamdb.info/app/244850/)

## About the Torch Server

**Torch** is the successor to SE Server Extender and gives server admins the tools they need to keep their Space Engineers servers running smoothly. It features a live management UI and a plugin system so you can configure your server exactly as you need. This egg runs Torch under Wine.

> ⚠️ **Do NOT cancel the first start!** If the first startup is interrupted you will need to delete and recreate the server from scratch.
